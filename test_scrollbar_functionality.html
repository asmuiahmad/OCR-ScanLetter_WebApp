Cara Menguji Compact Scrollable Containers:

  1. Fixed Height: Perhatikan container memiliki tinggi tetap dan tidak bertambah tinggi
  2. Scroll Test: Scroll ke bawah untuk melihat semua items - scrollbar selalu ada
  3. Compact Design: Container tidak memakan banyak ruang vertikal di halaman
  4. Hover Effect: Hover mouse pada scrollbar untuk melihat efek perubahan warna
  5. Responsive Test: Ubah ukuran browser - tinggi tetap kompak di semua device
